# Next-JS-App Template v1.0.0
GDSC NIT Silchar's Template for a NextJS + SASS app.

## Features, Tools and Technologies
- JavaScript
- NextJS 13
- pages router
- SASS Modules
- pnpm
- Prettier
- Auto Lint and Pretty on commit using simple-git-hooks (compatible with Windows unlike husky)
